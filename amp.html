Instalar Apache para Windows

Paso 1

Es recomendable tener todo ordenado, asi que antes de empezar va a crear la carpeta llamada Server, en la unidad raiz "C:\"

Paso 2

Se requiere de la ultima version adecuada de Apache, la actual version es "httpd-2.2.19-win32-x86-openssl-0.9.8r"

Paso 3

Despues de descargar el Apache, se inicia con la instalacion, se aceptan los terminos y condiciones, el programa de instalación reuiere de algunos datos: Network Domain, Server Name, Email Address.

Paso 4

En la siguiente ventana tienes que seleccionar los componentes y la ubicacion para que sean accesibles, “C:\Server\Apache\” > Next e Install.





Instalar PHP para Windows

Paso 1

Descargar la version binaria desde el sitio oficial de PHP, "http://windows.php.net/download/"

Paso 2

Descomprimir el contenido del archivo zip en un directorio de eleccion, ejemplo: "C:\PHP" e iniciar la instalación

Paso 3

Despues de instalar el producto, copie el archivo php.ini-produccion en php.ini, en la misma carpeta.

Paso 4

Las directivas son relevantes para la configuracion de PHP, extension_dir, extension, log_errors, error_log, cgi.force_redirect, cgi.fix_pathinfo.
